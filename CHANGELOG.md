# Changelog

All notable changes to FathomSeed will be recorded here.

## Unreleased

### Added

- AGPL-3.0 license and separate commercial licensing notice
- Learning plan flow with a no-provider fallback
- FastAPI backend with SQLite and SQLAlchemy
- Vue 3 + Tailwind frontend
- Goal clarification, strategy selection, and learning method selection
- Multi-mode learning task generation
- Dedicated activity surfaces for drills, learning games, podcasts, flashcards, projects, mentor dialogue, and other modes
- Local exercise providers
- Feedback events and dynamic plan pivoting
- Optional LLM model registry and routing layer
- Offline feedback queue concept for PWA usage

### Notes

- The product name is moving from FullMind to FathomSeed.
- The no-provider fallback remains available.
- Closed-source commercial use requires a separate license.
